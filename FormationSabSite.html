      • Vous personnalisez Menus

        Ce panneau est utilisé pour gérer les menus de navigation pour le contenu que vous avez déjà publié sur votre site. Vous pouvez créer des menus et y ajouter des éléments pour le contenu existant, comme les pages, les articles, les catégories, les étiquettes, les formats, ou des liens personnalisés.

        Les menus peuvent être affichés dans les emplacements définis par votre thème ou dans des zones de widget en ajoutant un widget de « Menu de navigation ».

      • Personnalisation Réglages de la page d’accueil

        Vous pouvez choisir ce qui est affiché sur la page d’accueil de votre site. Il peut s’agir d’articles dans l’ordre chronologique inversé (blog classique), ou une page fixe/statique. Pour définir une page d’accueil statique, vous devez au préalable créer deux pages. Une qui deviendra la page d’accueil, et une autre qui sera celle où vos articles seront affichés.

      • Personnalisation CSS additionnel

        Ajouter votre propre code CSS ici pour personnaliser l’apparence et la mise en page de votre site. En savoir plus sur le CSS (en anglais) (ouvre un nouvel onglet)

        Lorsque l’on utilise un clavier pour naviguer :

        • Dans la zone de modification, la touche Tab saisit un caractère de tabulation.
        • Pour partir de cette zone, utiliser la touche ESC puis la touche TAB.
        • Personnes utilisant un lecteur d’écran : en mode formulaire, vous aurez besoin d’utiliser la touche ESC deux fois.

        Le champ de modification met automatiquement en surbrillance la syntaxe du code. Vous pouvez désactiver ceci dans votre profil (ouvre un nouvel onglet) pour travailler en mode texte brut.
